html5에서 추가된 다양한 속성들

novalidate : 문법검사 하지 않음
accept-charset : 전송할 데이타를 지정한 문자코드셋으로 변환
disabled : 사용할 수 없는 상태로 지정
list : 목록힌트 제공
max / min : 최소값과 최대값지정
step : 단계 지정
pattern : 문법지정 - 참고) http://www.w3schools.com/js/js_regexp.asp
placeholder : 입력힌트 지정
autofocus : 자동으로 포커스를 가지게 될 요소에 지정
required : 필수입력항목 지정
autocomplete : 자동완성 on / off


몇가지 속성을 확인해보면 된다
disabled="disabled"을 사용하면 사용하지 않을 수 있다. 

autofocus 는 처음에 커서가 깜빡거리는 위치를 지정한다
autofocus="autofocus"

required="required" 는 필수 입력항목을 지정하는데 사용한다. 

autocomplete="off"
on, off 를 통해서 자동완성 기능의 사용을 조정한다
기본이 on 이다.
