HTML4 -> HTML5로 넘어오면서 바뀐 TABLE 속성

이제는 summary를 사용하지 않고 caption 사용하게끔 바뀜

서머리 요소 삭제되었으며, 굳이 표현을 하고 싶다면 details 태그를 사용하여 그 안에 summary 태그와 내용을 작성한다. HTML5 에서는 summary 는 넣을 수 없는 환경이므로 사용하지 않는다. 삭제된 이유 시각 장애인분들 이외에는 모호한 태그 과도한 친절을 넘어서 방해로 분류되었으므로 제거가 됨 caption 요소만 넣는 것이 좋다.
