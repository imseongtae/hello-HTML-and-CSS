HTML 에 대한 견해

웹의 철학을 실천하고 싶습니다.
웹의 접근성을 높이기 위해서 HTML 태그의 적절하고 정확한 사용을 중요하게 생각합니다. 웹 페이지의 본질은 문서인 까닭에 더 명확한 내용을 전달하기 위해 구조적인 설계에 힘씁니다. 어떠한 장애에도 구애 없이 정보가 전달될 수 있도록 지속해서 공부합니다.

CSS 에 대한 견해

CSS 를 표현하는 장인이 되고 싶습니다.
CSS는 HTML을 아름답게 꾸며주는 언어입니다. 화가가 붓을 만드는 법을 알 필요는 없지만, 붓을 만드는 사람보다 붓 질은 훨씬 잘해야 할 것입니다. 붓 질하는 법을 모른다면 표현하고자하는 것을 제대로 표현하는 것은 어려운 일이 됩니다. CSS는 디지털 컨텐츠를 생산하는 사람들에게는 붓 질과 같은 것이라고 생각합니다.

Javascript 에 대한 견해

사용자 경험을 높이기 위해 고민합니다.
웹을 이용하는 이용자에게 잘 갖추어진 UI를 제공하는 것을 넘어서서 높은 수준의 사용자 경험(UX)을 제공하기 위해서는 ‘상호작용’이 중요하다고 생각합니다. 웹개발자의 Javascript 이해에 따라 웹에서 제공할 수 있는 사용자 경험이 달라질 수 있다는 점을 생각해보면서 Javascript의 중요성을 더욱 실감했습니다.
